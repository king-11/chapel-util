ChapelEnv¶

Note

All Chapel programs automatically use this module by default. An explicit use statement is not necessary.

Access to Chapel Environment Variables

The values of Chapel’s environment variables upon compile time are accessible through the built-in parameters shown below. This information can also be displayed from the command line by executing the compiled program with the --about flag.
