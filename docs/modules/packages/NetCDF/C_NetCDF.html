C_NetCDF¶

Usage

use NetCDF.C_NetCDF;

or

import NetCDF.C_NetCDF;

The NetCDF module defines the interface to the netCDF library. Documentation for its functions, types, and constants can be found at the official netCDF web site: https://www.unidata.ucar.edu/software/netcdf/docs
