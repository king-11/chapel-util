Usage

use HDF5.C_HDF5;

or

import HDF5.C_HDF5;

The C_HDF5 module defines the interface to the HDF5 library. Documentation for its functions, types, and constants can be found at the official HDF5 web site: https://portal.hdfgroup.org/display/HDF5/HDF5
